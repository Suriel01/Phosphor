Phosphor:
    - high-level abstraction language
    - contains:
        - classes
        - enums
        - structures
        - aritmetics
    - all variable types are dynamically typed but there will be "var" specificator which stands for no specified data typed
    - language will be as simple as it can be (syntax will look similar to your favourite book chapter) 



TOKENS:

*aritmetic operations:
    + //addition
    - //substraction
    * //multiplication
    : //division
    % //modulo
    ^ //power

*data types:
    STRING //character literal
    CHAR
    BOOLEAN //true,false,unpredictable
    INTEGER
    SHORT
    BIT
    FLOAT
    DOUBLE

g++ -W -Wall -s *.cpp -o [name]
